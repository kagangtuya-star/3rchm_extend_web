学派大师

学派大师 Master Specialist

“你所学习的魔法都有着种种规则和限制，只有了解他们的本质，你才能学会如何用最好方法对付他们。”——卡普德尔·波兰达，惑控学派创始人

 

许多法师都选择了专攻某个学派，以强化这个学派的法术。学派大师也需要专攻某个学派，但她全身心地投入研究，让她比普通专精法师拥有更深程度的理解

如果说专精法师是学派的学生，那么学派大师便是学派的教师

成为一名学派大师：要成为学派大师必须要有法师的学派专精职业能力——其他奥术施法者根本就无法满足这个进阶的先决条件。所以要成为学派大师最直接也几乎是唯一的方法就是成为一名专精法师，你可以在法师等级上获得3级，然后在4级为进阶学派大师，当你提升完学派大师的10级到13级后，你可以继续提升法师等级，或是考虑另一个进阶职业，例如大法师。

学派大师所描述的“所选学派”这个词，意指着你在专精法师上专精的那个学派。

特殊：必须是一名专精法师

技能：知识（神秘）５级，法术辨识５级　

专长：法术专攻（你的专精学派）

职业能力

每日法术／可知法术：每一级你都获得新的法师每日法术量，就像你在法师等级上升了一级一样。但你并不获得任何其他的法师职业给予你的好处（如奖励专长）。　

技能专攻（法术辨识）：１级时你获得技能专攻（法术辨识）作为奖励专长。　

扩展法术书：当你到达２级时，你可以选择一个你专精学派的法术加入你的法术书。该法术可以是你可以施展的任意等级法术，这是每次升级获得的２个法术以外额外奖励的法术。　

５级和８级时，你可以将另外一个你专精学派的法术加入你的法术书。　

高等法术专攻：３级时，你获得你专精的学派的高等法术专攻作为奖励专长。　

学派之秘：　

l  次级学派之秘：４级开始，你初次获得了你一直专精于某一学派的法术带来的第一个好处。你依据你的专精学派获得以下能力：　

n  防护：你在解除法术检定上获得１／２学派大师等级的表现加值。　

n  咒法：所有你召唤或呼唤的生物获得额外的等于你学派大师等级的生命值。　

n  预言：你施展的预言法术如果只能在专注中生效，当你结束专注后，它将继续生效１／２你学派大师等级轮。你可以在结束这个专注后施展其他法术，仍然不会失去上一个法术的效果。　

n  惑控：你的魅惑目标将不获得额外的豁免加值，即使他们正在被你或者你的同伴威胁或者攻击。另外，你的胁迫目标在胁迫下做出违反天性的事情时，也不获得额外的豁免加值。　

n  塑能：当施展塑能系法术时，你的专注检定获得１／２你学派大师的加值。　

n  幻术：你所有的具有“意志，通过则不相信”描述的幻术系法术的豁免难度增加２　

n  死灵：当你施展死灵系法术时，周围６０尺所有的不死盟友获得等同于你的学派大师等级的驱散抗力和豁免加值，这个效果持续你的学派大师等级轮。　

n  变化：当你的变化系法术被成功解除时，你的变化系法术仍然继续持续一轮。如果某个可信赖的生物受到了解除法术的效果，它将知道变化系法术将被移除但仍将持续额外一轮。　

l  中级学派之秘：７级开始，你将依照你的专精学派获得第二个好处。所有这些能力将在你施展本学派法术时自动触发，持续等同于你的学派大师轮。　

n  防护：如果对你释放的法术，你豁免成功将产生部分或者减半效果，你在豁免成功后将不受到任何效果。　

n  咒法：在对你的咒法系法术的解除魔法时，你被视为比你现在的施法者等级高５级。　

n  预言：你在预言系法术作用时，获得离奇闪避能力。　

n  惑控：你可以在对惑控或者影响心灵的法术或类法术能力的意志豁免检定失败后马上进行一次重投，你必须接受结果即使结果更糟。　

n  塑能：在你释放造成能量的法术之后，你获得你释放的法术的能量形式的２０点能量抗力。

n  幻术：你获得失手几率２０的隐蔽效果。　

n  死灵：你免疫属性伤害、吸取，能量吸取和负向等级。　

n  变化：你可以在对变化系法术或者类法术能力的强韧豁免检定失败后马上进行一次重投，你必须接受结果即使结果更糟。　

l  高级学派之秘：１０级时，你对你专精学派的研究已臻完美。你根据你的专精学派获得以下好处，每种能力每天可以使用３次。　

n  防护：当施展防护系法术时，如果法术只能对你本人使用，你可以将这个法术转化为接触法术对其他人使用。　

n  咒法：你可以以一个迅捷动作施展那些需要１个标准动作施展的咒法系法术。　

n  预言：当你施展预言系法术时，你获得５轮的“真知术”效果。　

n  惑控：所有在你的惑控法术的豁免检定中成功的生物，在成功之后第二轮，必须重新进行一次豁免（就像你重新施展了这个法术），但其豁免有＋５加值。　

n  塑能：所有在你塑能法术的豁免检定中失败的生物，将额外在下一轮时受到他豁免失败的法术的一半的伤害。　

n  幻术：你可以施展任何幻术系法术而无需姿势、语言和材料成分（就像搭配了默发、定发、施法免材专长一样），而不需要耗费更高的法术位。　

n  死灵：当你施展一个死灵系法术时，周围６０尺内所有不死盟友获得１０点快速医疗，持续５轮。变化：当某个生物成功的豁免了你的变化系法术，他将受到等同于于你施展的法术等级的伤害。　

施法者等级提升：６级时，当你施展你的专精学派法术时，你的施法者等级＋１，９级时这项加值升为＋２。
